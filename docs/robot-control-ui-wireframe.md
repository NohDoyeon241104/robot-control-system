## 1. 전체 화면 구조

### 1.1 레이아웃 구성
```
┌─────────────────────────────────────────────────────────────┐
│ Header (고정)                                                │
│ - 로고 | 프로젝트명 | 알림 | 사용자 프로필                    │
├─────────────┬───────────────────────────────────────────────┤
│             │                                               │
│             │                                               │
│  Sidebar    │         Main Content Area                    │
│  (네비게이션) │         (동적 라우팅 영역)                    │
│             │                                               │
│             │                                               │
│             │                                               │
├─────────────┴───────────────────────────────────────────────┤
│ Footer (선택적)                                              │
│ - 시스템 상태 | 연결 상태 | 버전 정보                         │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 색상 및 테마 가이드
- **Primary Color**: `#2196F3` (파란색 - 신뢰, 기술)
- **Secondary Color**: `#FF9800` (주황색 - 경고, 액션)
- **Success**: `#4CAF50` (녹색)
- **Danger**: `#F44336` (빨간색)
- **Background**: `#F5F5F5` (연한 회색)
- **Dark Mode**: `#1E1E1E` (다크 배경)

**주요 기능**:
- 실시간 로봇 상태 모니터링

- XSS 방지

## 7. 기술 스택 추천

### Frontend (Vue.js)
- **UI 프레임워크**: Vuetify 3 또는 Element Plus
- **차트**: Chart.js 또는 Apache ECharts
- **3D/캔버스**: Three.js (3D 시각화), Konva.js (2D 캔버스)
- **상태 관리**: Pinia
- **HTTP 클라이언트**: Axios
- **WebSocket**: native WebSocket API 또는 Socket.io-client

### Backend (Spring Boot)
- **WebSocket**: Spring WebSocket + STOMP
- **실시간 통신**: RSocket (대안)
- **DB**: PostgreSQL 또는 MySQL
- **ORM**: Spring Data JPA

### ROS2 연동
- **옵션 1**: rosbridge_suite (WebSocket 기반, 가장 간단)
- **옵션 2**: rcljava (Java Native)
- **옵션 3**: Custom Python Bridge + MQTT
